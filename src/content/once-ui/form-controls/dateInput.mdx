The `DateInput` component provides an enhanced date picker that combines an input field with a dropdown calendar interface. It allows users to easily select dates with optional time selection.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={<DateInputExample />}
  codes={[
    {
      code:
`const [date, setDate] = useState<Date | undefined>(new Date());
  
const handleDateChange = (newDate: Date) => {
  setDate(newDate);
};

<DateInput
  id="date-input-example"
  label="Select Date"
  value={date}
  onChange={handleDateChange}
/>`,
      language: "tsx",
      label: "DateInput"
    }
  ]}
/>

## With Time Picker

Enable time selection by setting the `timePicker` prop to `true`.

## Features

- **Calendar Interface**: Visual calendar for easy date selection
- **Time Selection**: Optional time picker for selecting hours and minutes
- **Formatted Display**: Displays selected date in a readable format
- **Dropdown Integration**: Uses DropdownWrapper for a clean UI experience
