The `Input` component provides a text entry field with floating labels, validation support, and customizable styling. Use it for collecting user input in forms.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Input
      id="basic-input"
      label="Username"
    />
  }
  codes={[
    {
      code:
`<Input
  id="basic-input"
  label="Username"
/>`,
      language: "tsx",
      label: "Input"
    }
  ]}
/>

## Validation

The Input component supports error states and validation messages.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Input
      id="error-input"
      label="Email"
      error={true}
      errorMessage="Please enter a valid email address"
    />
  }
  codes={[
    {
      code:
`<Input
  id="error-input"
  label="Email"
  error={true}
  errorMessage="Please enter a valid email address"
/>`,
      language: "tsx",
      label: "Validation"
    }
  ]}
/>

## With prefix and suffix

Add icons or other elements before or after the input text.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Input
      id="prefix-suffix-input"
      label="Search"
      hasPrefix={<Icon name="search" size="xs" />}
      hasSuffix={<IconButton icon="close" size="s" />}
    />
  }
  codes={[
    {
      code:
`<Input
  id="prefix-suffix-input"
  label="Search"
  hasPrefix={<Icon name="search" size="xs" />}
  hasSuffix={<IconButton icon="close" size="s" />}
/>`,
      language: "tsx",
      label: "Prefix & Suffix"
    }
  ]}
/>

## Label as placeholder

Use the label as a placeholder that disappears when the input is focused.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Input
      id="placeholder-input"
      label="Enter your name"
      labelAsPlaceholder
    />
  }
  codes={[
    {
      code:
`<Input
  id="placeholder-input"
  label="Enter your name"
  labelAsPlaceholder
/>`,
      language: "tsx",
      label: "Label as placeholder"
    }
  ]}
/>
