The InteractiveDetails component is a utility component used to display labels and descriptions for interactive form elements such as checkboxes and radio buttons. It provides consistent styling and behavior for these elements.

## Usage with form controls

InteractiveDetails is primarily used as a building block for form controls like Checkbox and RadioButton to provide consistent labeling and description functionality.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Column gap="16">
      <Checkbox 
        label="Checkbox with InteractiveDetails"
        description="This checkbox uses InteractiveDetails for its label and description"
      />
      <RadioButton 
        label="Radio button with InteractiveDetails"
        description="This radio button uses InteractiveDetails for its label and description"
      />
    </Column>
  }
  codes={[
    {
      code:
`<Checkbox 
    label="Checkbox with InteractiveDetails"

/>

<RadioButton 
    label="Radio button with InteractiveDetails"
    description="This radio button uses InteractiveDetails for its label and description"
/>`,
      language: "tsx",
      label: "Form controls"
    }
  ]}
/>
