## Style your app
        
Once UI uses data attributes to set color profiles, border radius style, themes, and other styling options. Data modes are inherited from the parent and can be applied to any
semantic HTML element, providing simple, seamless style overrides.

## Override styles

Since data modes are inherited, it means you can nest them inside one-another, allowing
you to have a dark-mode app with a light-mode header for example, or use rounded buttons
in particular places in a generally sharp interface.

Just add the `data-theme`, `data-neutral`, `data-brand`, `data-accent`, `data-border`, `data-surface`, `data-solid`, `data-solid-style`, `data-transition` or `data-scaling` attributes to any semantic HTML element to override its default style.
