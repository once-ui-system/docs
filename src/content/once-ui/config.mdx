Once UI uses data attributes to set brand-, accent-, neutral color schemes, border radius
style, and light- and dark mode. Data modes are inherited and can be applied to any
semantic HTML element, providing easy customization and styling options to minimize the
size of tokens and styles.

## Default theme

Set the default style of your application in the{" "}
`resources/config.js` file.

Add the `data-theme`, `data-neutral`, `data-brand`, `data-accent`, `data-border`, `data-surface`, `data-solid`, `data-solid-style`, `data-transition` or `data-scaling` attributes to any semantic HTML element to override the default style.

## Nested themes

Data modes work with inheritance, meaning you can nest them inside one-another, allowing
you to have a dark-mode app with a light-mode header for example, or use rounded buttons
in particular places in a generally sharp interface.
