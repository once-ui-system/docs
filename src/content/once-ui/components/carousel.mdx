An interactive component for displaying a collection of images with smooth transitions and navigation controls. The Carousel supports both line and thumbnail indicators for navigation between images.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Carousel
      radius="l"
      items={[
        { slide: "/images/docs/once-ui/vibe-coding-light.jpg", alt: "Vibe coding light theme" },
        { slide: "/images/docs/once-ui/vibe-coding-dark.jpg", alt: "Vibe coding dark theme" },
      ]}
    />
  }
  codes={[
    {
      code:
`<Carousel
    items={[
      { slide: "/images/1.jpg", alt: "Image 1" },
      { slide: "/images/2.jpg", alt: "Image 2" },
    ]}
/>`,
      language: "tsx",
      label: "Carousel"
    }
  ]}
/>

## Indicator

The Carousel supports two types of indicators for navigation: line (default) and thumbnail.

## Custom content
