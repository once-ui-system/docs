A component for displaying multiple user avatars in a compact, horizontal stack. Commonly used in collaborative interfaces to show team members, participants, or contributors.

## Limiting displayed avatars

Use the `limit` prop to control how many avatars are displayed. Any additional avatars will be represented by a count indicator.

## Reverse order

Use the `reverse` prop to change the stacking order of avatars.

<CodeBlock
  marginTop="8"
  marginBottom="16"
  codePreview={
    <Column fillWidth gap="24">
      <AvatarGroup
        avatars={[
          { value: "A" },
          { value: "B" },
          { value: "C" },
          { value: "D" }
        ]}
        size="m"
        reverse={false}
      />
      <AvatarGroup
        avatars={[
          { value: "A" },
          { value: "B" },
          { value: "C" },
          { value: "D" }
        ]}
        size="m"
        reverse={true}
      />
    </Column>
  }
  codeInstances={[
    {
      code:
`// Default order (first avatar on top)
<AvatarGroup
  avatars={[
    { value: "A" },
    { value: "B" },
    { value: "C" },
    { value: "D" }
  ]}
  size="m"
  reverse={false}
/>

// Reversed order (last avatar on top)
<AvatarGroup
  avatars={[
    { value: "A" },
    { value: "B" },
    { value: "C" },
    { value: "D" }
  ]}
  size="m"
  reverse={true}
/>`,
      language: "tsx",
      label: "Reverse order"
    }
  ]}
/>
